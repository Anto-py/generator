Bienvenue !

Créez des dialogues pédagogiques multilingues avec synthèse vocale, en un clic.

Il vous suffit d'une clé API Google gratuite pour commencer.

  1. Rendez-vous sur aistudio.google.com
  2. Connectez-vous avec votre compte Google
  3. Cliquez sur « Get API Key » (en haut à gauche)
  4. Cliquez sur « Create API key in new project »
  5. Copiez la clé affichée (elle commence par AIza...)
  6. Collez-la dans le champ ci-dessus
🔒 Votre clé reste sur votre appareil. Elle n'est jamais partagée avec un serveur tiers.
